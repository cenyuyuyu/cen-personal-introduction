# 🧑 个人介绍页面

一个使用纯 HTML + CSS + JavaScript 构建的个人介绍静态页面，展示个人基本信息、学习计划和联系方式。

## 📸 预览

页面包含头像展示、自我介绍、学习时间线、联系方式等模块，整体风格简洁清新。

## ✨ 功能特性

| 功能 | 说明 |
| :--- | :--- |
| 📖 **查看更多/收起** | 点击按钮展开或收起更多个人介绍内容 |
| ⌨️ **打字机效果** | 页面加载后自动播放打字动画 |
| 🎯 **导航高亮** | 滚动页面时，当前所在区域的导航链接自动高亮 |
| 🌀 **平滑滚动** | 点击导航链接平滑滚动到对应区域 |

## 🗂️ 页面结构

```text
first/
├── index.html          # 页面结构
├── style.css           # 样式美化
├── script.js           # 交互功能
├── 1.jpg               # 头像图片
└── README.md           # 项目说明
```

## 🚀 快速开始

直接用浏览器打开 `index.html` 文件即可预览页面，无需安装任何依赖。

```bash
# 克隆仓库
git clone https://github.com/cenyuyuyu/cen-personal-introduction.git

# 进入项目目录
cd cen-personal-introduction/first

# 用浏览器打开
start index.html
```

## 📄 页面内容

- **个人介绍** — 姓名、专业背景、兴趣爱好
- **学习计划** — 按时间线展示四个阶段的学习规划：
  - 第一阶段：基础夯实（HTML5 / CSS3 / JavaScript）
  - 第二阶段：进阶提升（响应式 / ES6+ / 构建工具）
  - 第三阶段：框架实战（Vue.js / React）
  - 第四阶段：持续深入（TypeScript / 性能优化）
- **联系方式** — 邮箱、GitHub、电话

## 🛠️ 技术栈

- **HTML5** — 语义化标签结构
- **CSS3** — Flexbox / Grid 布局、过渡动画、平滑滚动
- **JavaScript** — DOM 操作、事件监听、定时器动画
