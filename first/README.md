# 个人介绍

这个网页主要是自己的一些介绍，以及相关的暑假计划和学习目标，还有联系方式。

## 页面内容

- 个人简介（姓名、专业、方向）
- 学习计划时间线（四个阶段）
- 联系方式（邮箱、GitHub、电话）

## 功能

- 查看更多/收起
- 打字机效果
- 导航高亮
- 平滑滚动

## 文件结构

```text
index.html  页面结构
style.css   样式
script.js   交互功能
1.jpg       头像
```

## 使用方式

直接用浏览器打开 index.html 即可。

## 技术栈

HTML5 / CSS3 / JavaScript
