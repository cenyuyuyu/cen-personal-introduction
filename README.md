# 个人介绍页面

一个使用纯 HTML + CSS + JavaScript 构建的个人介绍页面。

## 页面内容

- **头像** — 以姓名首字母展示的圆形头像
- **基本信息** — 姓名、专业、方向
- **关于我** — 个人简介（可展开/收起）
- **技能** — 带进度条的技能卡片（HTML / CSS / JavaScript / Git）
- **学习计划** — 时间线展示四个阶段的学习规划
- **联系方式** — 邮箱、GitHub、所在城市

## 交互功能

| 功能 | 说明 |
|------|------|
| 🌗 主题切换 | 点击页脚的按钮可切换暗色/亮色主题，偏好会被保存 |
| 📖 查看更多 | 点击"查看更多"按钮可展开或收起更多个人介绍 |
| 📊 技能条动画 | 滚动到技能区域时，进度条会自动填充动画 |
| ⬆️ 回到顶部 | 向下滚动超过 400px 时，右下角会出现回到顶部按钮 |
| 🎯 平滑滚动 | 点击导航链接会平滑滚动到对应区域 |

## 运行方式

直接用浏览器打开 `index.html` 文件即可预览页面。

## 技术栈

- HTML5
- CSS3（Flexbox / Grid / 渐变 / 动画 / 响应式）
- JavaScript（DOM 操作 / 事件监听 / localStorage）

## 项目结构

```
├── index.html    # 页面结构
├── styles.css    # 样式美化
├── script.js     # 交互功能
└── README.md     # 项目说明
```
